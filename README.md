# Desktop-File-Sorter
A Python-based desktop automation tool that automatically organizes files into folders based on file type.
## Features
- Sorts Documents, Videos, PNG files, LNK, ZIP Files and Archives
- Automatic folder creation
- Easy-to-use interface
- Reduces desktop clutter
## Technologies Used
- Python
- OS Module
- Shutil
- Jupyter Notebook
## How to Run
```bash
python file_sorter.py
